# tume.js
長体にして文字を詰めるjQuery拡張。

widthを指定したブロック要素に対して使う。

##使い方

    usage $(hoge).tume() innerHTMLを詰める
          $(hoge).tume(str) strを詰めて設定
          $(hoge).tume(str,true) widthに満たない場合は広げる
